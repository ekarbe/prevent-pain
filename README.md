# Prevent Pain

A web application designed to help users identify potential causes of their physical pain and provide suggestions for strengthening exercises. Users can click on an interactive anatomy map to select an area of pain, which then guides them through possible causes and recommended rehab plans.

## ⚠️ Work in Progress

This project is currently under active development. The content is for demonstration purposes only and serves as a proof of concept. It is **not** a substitute for professional medical advice. If you have acute, sharp pain, or cannot walk, please see a doctor immediately.

## Key Features

*   **Interactive Anatomy Map**: Click on a body part to begin diagnosis.
*   **Guided Cause Selection**: Choose from a list of potential causes for your pain.
*   **Exercise Recommendations**: Get a list of exercises tailored to the selected cause.
*   **Responsive Design**: Works on both desktop and mobile devices.

## Tech Stack

*   **Frontend**: [React](https://reactjs.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Icons**: [Lucide React](https://lucide.dev/)
*   **Build Tool**: [Vite](https://vitejs.dev/)

## Getting Started

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd prevent-pain
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
